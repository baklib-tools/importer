## 文档阅读顺序（建议从这里开始）

本项目用于把“海量文件路径清单”整理成可协作填写的 Excel，并通过 Baklib Open API 批量导入 DAM（并可选创建站点页面）。

按下面顺序阅读与执行即可跑通全流程：

1. `01-workflow-sop.md`：端到端 SOP（推荐先通读一遍，建立全局理解）
2. `02-file-list-mac-linux.md`：macOS / Linux 生成路径清单（Windows 见排障文档中的相关段落）
3. `03-excel-guide.md`：Excel 列含义与填写规范（业务协作填写的关键）
4. `04-import-quickstart.md`：导入脚本快速开始（最短路径跑通一次导入）
5. `05-import-runbook.md`：导入脚本“行为与参数”详解（需要精细控制时再查）
6. `06-windows-troubleshooting.md`：Windows 常见问题与排障
